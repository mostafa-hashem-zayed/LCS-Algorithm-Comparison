# 🚀 Longest Common Subsequence (LCS) Algorithm Comparison

An interactive web application that compares three approaches for solving the Longest Common Subsequence (LCS) problem: **Brute Force**, **Recursive (Memoization)**, and **Dynamic Programming**. The project provides real-time performance analysis, DP table visualization, and complexity comparison through a modern and responsive interface.

## ✨ Features

- Compare three LCS algorithms
- Real-time execution time analysis
- Dynamic Programming table visualization
- Complexity comparison
- Interactive user input
- Responsive and modern UI

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript

## 📈 Algorithms

- Brute Force
- Recursive (Memoization)
- Dynamic Programming
