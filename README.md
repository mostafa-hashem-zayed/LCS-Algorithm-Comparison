# 🧩 Longest Common Subsequence (LCS) Algorithm Comparison

An interactive web application that compares three different approaches for solving the **Longest Common Subsequence (LCS)** problem: **Brute Force**, **Recursive (Memoization)**, and **Dynamic Programming**. The project provides real-time execution time analysis, algorithm complexity comparison, and Dynamic Programming table visualization through a modern and responsive interface.

---

## 📖 Overview

This project was developed to demonstrate and compare multiple algorithms used to solve the Longest Common Subsequence (LCS) problem. It allows users to enter two input strings and instantly compare the execution time and performance of each algorithm while visualizing the Dynamic Programming solution.

---

## ✨ Features

- 🔍 Compare three LCS algorithms
- ⚡ Real-time execution time analysis
- 📊 Dynamic Programming table visualization
- 📈 Time & Space complexity comparison
- ✍️ Interactive user input
- 📱 Responsive and modern user interface
- 🎯 Educational algorithm visualization
